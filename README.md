bc4py poc plotter
====
This is PoC hash plot tool.

Requirements
====
* Rust nightly [Install](https://forge.rust-lang.org/other-installation-methods.html#more-rustup)

How to use
====
```bash
git clone https://github.com/namuyan/bc4py_plotter
cd bc4py_plotter
cargo build --release
cp target/release/bc4py_plotter .
./bc4py_plotter
```
